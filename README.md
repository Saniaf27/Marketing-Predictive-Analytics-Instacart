# Marketing Predictive Analytics Project

## Title
Predicting Product Reorders using Machine Learning

## Domain
Marketing Analytics

## Problem Statement
Predict whether a customer will reorder a product based on their previous purchasing behaviour.

## Dataset
Instacart Market Basket Analysis Dataset

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- LightGBM

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Best Model
Histogram Gradient Boosting

The complete Instacart dataset is publicly available on Kaggle. Due to GitHub's file size limit, only the smaller reference files are included in this repository.
